\textbf{任务1.2: } 了解数据

你会接触到各种各样的数据表，因此在读取之后，我们有必要通过一些简单的方法，来了解我们数据表是什么样子的。

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  获取数据表的行列，并打印。
\item
  使用 \texttt{.head()}、\texttt{.tail()}、\texttt{.sample()}
  方法，观察、了解数据表的情况。
\item
  使用 \texttt{.dtypes} 属性，来查看各列数据的数据类型。
\item
  使用 \texttt{isnull()} 配合 \texttt{.any()}
  等方法，来查看各列是否存在空值。
\item
